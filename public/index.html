ZenTao Automation Hub

禅道自动化测试与 Bug 提交系统

这个版本已经支持真实禅道接入。优先建议使用 OpenAPI Token；如果你们实例较老或未开启 OpenAPI，可以切换到 Session 登录模式。

当前接入说明

  • OpenAPI 模式会调用 `/api.php/v1/products/:productId/bugs`。
  • Session 模式会先请求 `api-getsessionid.json`，再走 `user-login.json` 和 `bug-create-*.json`。
  • 你给的实例地址已预填为 `https://zt.codetech.pro/zentao`。
